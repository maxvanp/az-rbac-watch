# diff

Compare two RBAC snapshots and report changes.

The diff report highlights added, removed, and modified assignments between two JSON snapshots generated by `snapshot`.

## Usage

```bash
az-rbac-watch diff old-snapshot.json new-snapshot.json
```

## Options

| Option | Short | Description |
|--------|-------|-------------|
| `OLD_SNAPSHOT` | | Path to the older snapshot JSON file |
| `NEW_SNAPSHOT` | | Path to the newer snapshot JSON file |
| `--format FORMAT` | `-f` | Output format: `console`, `json`, or `html` |
| `--output PATH` | `-o` | Output file path |

HTML output is also auto-detected when `--output` ends with `.html` and `--format` is left at its default.

## Examples

```bash
# Console output
az-rbac-watch diff snapshot-old.json snapshot-new.json

# JSON for automation
az-rbac-watch diff snapshot-old.json snapshot-new.json --format json -o changes.json

# HTML report
az-rbac-watch diff snapshot-old.json snapshot-new.json --output changes.html
```

## Exit codes

| Code | Meaning |
|------|---------|
| `0` | No RBAC changes detected |
| `1` | Changes detected |
| `2` | Error — missing file, invalid snapshot, or invalid arguments |
